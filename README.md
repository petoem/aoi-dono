# Aoi-dono

Post from your favourite text `$EDITOR` to Bluesky and Mastodon. Simply run `aoi-dono` in your shell, write your post, save it and you are done!

## How to get `Aoi`?

```sh
go install github.com/petoem/aoi-dono
```
